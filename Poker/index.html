Poker & Casino

This is by far the biggest and best project I’ve created so far. It’s a full-stack application featuring a working database for user profiles and real-time leaderboards. The backend handles traffic, user creation, and profile editing. It also includes a Poker AI you can play against in heads-up matches, functioning multiplayer to compete with friends online, competitive slots, and a variation of Texas Hold'em called Kuhn Poker.

I started this project out of my love for poker. Initially, I just wanted to build a Poker AI, but it evolved into something my friends and I could enjoy together, challenging each other to beat high scores. This project taught me so much about web development, and it’s been incredibly rewarding to see it all come together.
