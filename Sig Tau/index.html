Sig Tau Sign Up

Learning Express alongside Node.js was the driving force behind this project. I wanted to create a site for my fraternity that would give new members a distinct and engaging platform to visit during sign-ups. The site features a main landing page, an editing page, and a database viewing page.

The database uses a very rudimentary plain text document stored on the server, which is sufficient for this project since it only handles a small number of entries. The editing functionality allows the fraternity to customize the images and colors on the page, depending on the theme of the rush event. While simple, this project was a great way to practice backend development and learn how to build dynamic, user-editable content.
